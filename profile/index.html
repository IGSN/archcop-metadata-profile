    Overview of Version 1.0

    Cite As

    IGSN–DataCite Archaeology and Cultural Heritage Community of Practice. (2025). Archaeological Sample Metadata Profile. Version 1.0. IGSN e.V. https://doi.org/10.5281/zenodo.17254975

    Version 1.0 is the first published version of the Archaeological Sample Metadata Profile. It has been released alongside Version 1.0 of the IGSN–DataCite Archaeology CoP Crosswalk Recommendation that translates the Metadata Profile into DataCite Metadata Schema V4.6 for the purposes of registering IGSN IDs for archaeological samples.
