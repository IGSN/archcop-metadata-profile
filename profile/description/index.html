    8. Description

    Obligation: Recommended

    Definition:
    Detailed description of the sample. Should include information about the sample and its collection not captured in other properties. For example, the primary reason for sample collection in terms of analytics.

    Allowed values, examples, other constraints:
    1 bag of soil: dark greyish brown with slate, glass, brick, plaster inclusions
