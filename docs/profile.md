Version 1.0 is the first published version of the Archaeological Sample Metadata Profile. It has been released alongside Version 1.0 of the [IGSN–DataCite Archaeology CoP Crosswalk Recommendation](https://doi.org/10.5281/zenodo.17446558) that translates the Metadata Profile into [DataCite Metadata Schema V4.6](https://schema.datacite.org/meta/kernel-4.6/) for the purposes of registering IGSN IDs for archaeological samples.
